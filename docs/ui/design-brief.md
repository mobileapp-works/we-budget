# デザインブリーフ（WE BUDGET）

Claude Design（claude.ai/design）でモックアップを作る際のインプット。確定したビジュアルは `docs/ui/*.md` のルールに反映する。

## 概要
- アプリ名: WE BUDGET
- ジャンル: ライフスタイル / ファイナンス（カップル家計簿）
- 世界観: **ポップ＆あたたかい**。親しみやすく、2人で楽しく続けられる空気感。やや角丸・余白広め
- カラー方向性: **メイン=コーラル/オレンジ系**、**アクセント=ターコイズ/ティール**
- 背景: 白系（ライト）/ ニュートラルダーク（ダーク）。**両対応・端末設定に追従**
- ターゲット: 同棲・夫婦・恋人など2人で家計管理したいカップル全般（年齢幅広め、一般的スマホユーザー）
- 参考アプリ: 特になし（おまかせ。OsidOriのやわらかさ＋B/43の見やすさを意識）
- iPad対応: あり（〜700px中央寄せ、リストはグリッド化）

## トーン
- 数字（お金）は読みやすく明快に。コーラルは「使いすぎ警告」ではなく「ポジティブ・あたたかい」基調色として使う
- 立替・精算など緊張しがちな機能を、やわらかい配色とコピーで角を取る
- 色だけに頼らず、アイコン＋テキストで状態を伝える（アクセシビリティ）

## Claude Design に貼るプロンプト（コピペ用）

```
以下のモバイルアプリ（iOS, React Native/Expo）のUIデザインを作成してください。

【アプリ概要】
- アプリ名: WE BUDGET
- ジャンル: カップル向け家計簿（2人で支出を記録し、立替精算・見える化）
- 世界観: ポップ＆あたたかい、やや角丸、余白広め、親しみやすい

【カラー方針】
- メイン: コーラル/オレンジ系（あたたかいが、警告ではなくポジティブ基調）
- アクセント: ターコイズ/ティール
- ライト/ダーク両対応、端末設定に追従
- コントラストは WCAG AA（4.5:1）を必ず満たす

【主要画面（モックアップしてほしい）】
1. ホーム — 今月の支出合計、立替残高カード、予算進捗バー、直近の支出リスト、上部に通知ベル
2. 支出入力（モーダル）— 「レシートで入力」/「手動で入力」の選択、金額・カテゴリ・支払い者（自分/相手/共同口座）・日付・メモ
3. 立替精算 — 立替残高（誰が誰にいくら）、精算ボタン、精算履歴
4. レポート — カテゴリ別の円グラフ・棒グラフ、期間切替（月/週/カスタム）
5. 設定 — 言語切替・ダークモード・通知ON/OFF・プライバシーポリシー・バージョン・ログアウト・アカウント削除

【デザイン要件】
- 8ptグリッド、タップ領域44pt以上
- iPhone SE〜iPad対応（レスポンシブ）
- 画面下部にバナー広告（高さ50-60px）を確保したレイアウト
- 各画面の4状態（Loading=スケルトン / Empty=説明+CTA / Error=再試行 / Success）

【出力してほしいもの】
1. 主要画面モックアップ（iPhone + iPad）
2. カラーパレット（ライト/ダーク両方、HEX）
3. タイポグラフィスケール
4. コンポーネント一覧（ボタン/カード/入力/リスト/バッジ/チップ）
5. 各画面の4状態
```
